# Case Studies

This directory contains case studies from Specula Method applications. Each study focuses on methodological learnings, challenges, and insights rather than just results. Some cases are anonymized to protect client confidentiality.

## Principles for Case Studies

### Anonymization & Confidentiality
- Client identity protected unless explicit permission granted
- Specific metrics generalized
- Context provided without identifying details
- Pilot projects noted as foundational to method development

### Focus on Method
- How phases were applied
- What worked and what didn't
- Adaptations made to standard process
- Insights for future applications

### Honesty About Limitations
- Failures and struggles documented
- Unexpected challenges highlighted
- Areas where method fell short

## Contributing Case Studies

If you've applied Specula Method and want to share learnings:

1. Follow the template structure in existing studies
2. Ensure thorough anonymization if needed
3. Focus on methodological insights
4. Include what didn't work, not just successes
5. Explicitly note if this is a pilot project that shaped method development
6. Submit via pull request
